# Umbraknight
Управление:

Идти: A/D
Бежать: Shift+A/D
Прыжок: Spase
Сесть: Ctrl

Инвентарь: B
Панель заклинаний: K

Для активации навыков необходимо переместить их на панель. По умолчанию первый слот панели привязан к клавише X.
